The K.i.D

About the Project

The K.i.D is an experimental electronic music project that merges dubstep, metal, and trance.

Behind the Sound

Each track is created using heavy synth layers, live drums, and distorted guitars. The goal is to create an experience that hits like an industrial sledgehammer.
